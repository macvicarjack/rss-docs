📞 Double-Dial Logic

1️⃣ Call 1 - Detection

If voicemail detected → hang up immediately

If live answer → proceed with conversation

2️⃣ Wait 5 Seconds

Brief pause before second attempt

3️⃣ Call 2 - Leave Message

If voicemail → leave pre-recorded message

If live answer → proceed with conversation

🎤 Sarah's Scripts

First Message (Live Answer)

"Hi, this is Sarah from [YOUR_COMPANY_NAME]. I am calling because we had you down for a roof estimate and wanted to make sure everything is okay. Did something come up?"

Voicemail Message

"Hey, this is Sarah from [YOUR_COMPANY_NAME]. We missed you earlier and wanted to reschedule your free roof estimate. I have openings today and tomorrow. I will text you now to grab a time."

🏷️ Stop Tags (Cadence Control)

When any of these tags are present, the cadence workflow stops.

engaged

Customer answered and talked on a call

Added by: Call Result Router
appointment_booked

Appointment was successfully booked

Added by: Call Result Router
booked

Alternative booking tag

Added by: Manual / GHL Automation
won

Deal closed successfully

Added by: Rep Won/Lost Check
lost

Deal was lost

Added by: Rep Won/Lost Check
stop_cadence

Manual force stop of any cadence

Added by: Manual
do_not_contact

Customer requested no further contact

Added by: Inbound Reply Handler
