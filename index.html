📞 Double-Dial Logic

1

Call 1 - Detection Only

If voicemail detected → hang up immediately. If live answer → proceed with conversation.

2

Wait 5 Seconds

Brief pause before second attempt.

3

Call 2 - Leave Message

If voicemail → leave the pre-recorded message. If live answer → proceed with conversation.

Stop Tags (Cadence Control)

When any of these tags are present, the cadence workflow stops.

engaged

Customer answered and talked on a call

Added by: Call Result Router

appointment_booked

Appointment was successfully booked

Added by: Call Result Router

booked

Alternative booking tag

Added by: Manual / GHL Automation

won

Deal closed successfully

Added by: Rep Won/Lost Check

lost

Deal was lost

Added by: Rep Won/Lost Check

stop_cadence

Manual force stop of any cadence

Added by: Manual

do_not_contact

Customer requested no further contact

Added by: Inbound Reply Handler
